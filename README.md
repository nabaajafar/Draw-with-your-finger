# Draw-with-your-finger

Browser-based AI drawing tool that uses hand tracking to let you draw in the air using your finger
It uses MediaPipe Hands to detect your index finger in real-time through your webcam, allowing you to:
- ✍️ Draw in the air
- 🧽 Erase strokes
- 🔄 Clear the canvas

All of this runs directly in the browser — no installation required.

# How It Works

The webcam tracks your hand using MediaPipe.
The tip of your index finger becomes your virtual pen.

# 🎮 Controls
Key	Action
Shift	Draw
Ctrl	Erase by touching strokes
CapsLock	Erase gradually (last stroke)
Space	Clear the entire canvas
